# TanStack Table Carbon

This package provides a comprehensive TanStack Table integration with Carbon
Design System components, including a reusable library and extensive examples.

## Features

This example application showcases:

- **Basic Table** - Simple table implementation
- **Column Customization** - Dynamic column visibility and ordering
- **Custom Filters** - Advanced filtering with custom filter components
- **Custom Filters (No Accordion)** - Filters without accordion layout
- **Editable Cells** - Inline editing capabilities
- **Empty State** - Handling empty data scenarios
- **Expansion & Radio Sticky** - Row expansion with sticky columns
- **Filter Panel** - Side panel filtering
- **Loading State** - Loading indicators
- **Localized Table** - Internationalization support
- **Mixed Filters** - Combining different filter types
- **Mix Table** - Complex table combinations
- **Row Expansion** - Expandable rows
- **Server Side** - Server-side data handling
- **Server Side Column Customization** - Server-side column management
- **Table Sizes** - Different table size variants
- **Themed Table** - Theme switching and customization
- **Virtualization** - Virtual scrolling for large datasets

## Getting Started

### Installation

```bash
yarn install
```

### Development

```bash
yarn dev
```

### Build

```bash
yarn build
```

## Structure

```
src/
  lib/
    tanstackTable/          # TanStack Table component library
      components/
      hooks/
      utils/
      ...
  examples/                 # Example implementations
    BasicTable.jsx
    ColumnCustomization.jsx
    ...
  Example.tsx              # Main example showcase
  main.tsx                 # Application entry point
```

## Usage

The examples demonstrate how to use the TanStack Table library with Carbon
Design System. Each example is self-contained and can be used as a reference for
implementing similar functionality in your application.
